About

ScratchPost is a project created by @redspacecat (Scratch) (GitHub) that has lots of cool features.

Features:

  • Forum search
  • Forum leaderboards
  • Viewing a user's forum statistics
  • An API so you can use any of this data

Notes:

The post counts on this website may not be entirely accurate if a post has since gotten dustbinned. It should be pretty accurate though. (within ~10 posts)

The cumulative post count by category chart doesn't include dustbinned posts, so the count may be lower than expected.

New posts are added every few hours.

Info:

The site is open source on GitHub.
